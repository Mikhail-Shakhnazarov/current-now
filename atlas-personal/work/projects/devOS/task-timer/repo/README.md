# Project Timing Tool

A CLI tool for tracking time spent on projects, with timer management, reporting, and configuration.

## Installation

Clone the repo and install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the tool:

```bash
python -m src.task_timer <command>
```

Commands:
- `start <project_id>` - Start timing a project
- `stop` - Stop current timer
- `status` - Show current timer status
- `report` - Generate time report
